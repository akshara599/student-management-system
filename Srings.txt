Operation	Example	Description
s.size() or s.length()	s.length()	returns string length
s.empty()	s.empty()	checks if string is empty
s.append("abc")	"Hi".append("abc") → "Hiabc"	concatenates
s + s2	"Hi" + "There"	concatenation using +
